# Inicializando o git Local




Primeiro confira se o git está instalado:




```bash

git --version

```

# Segundo configurar usuario

```

git config --global user.name "arthur.scherer"

git config --global user.email "arthurscherer96@gmail.com"

```

```

$cd Documents
